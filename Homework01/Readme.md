# Problems

1. Реализовать функцию возведения числа а в степень b. a, b ∈ Z, сводя количество выполняемых действий к минимуму.

*Пример 1:*

    а = 3, b = 2
    Ответ: 9

*Пример 2:*
    
    а = 2, b = -2    
    Ответ: 0.25
    
*Пример 3:*

    а = 3, b = 0
    Ответ: 1

*Пример 4:*

    а = 0, b = 0    
    Ответ: 1

*Пример 5:*

Входные данные находятся в файле <code>input.txt</code> в виде:

    b 3        
    a 10

Результат нужно сохранить в файле <code>output.txt</code>
    
Ответ: 1000
